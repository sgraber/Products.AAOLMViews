New Views
=========

1. blogentry_view.pt:           News Item   View for News Item look like a blog entry
2. blog_view.pt:                Topic       View for Topic look like a blog listing
3. toc_view.pt                  Topic       View for Topic look like an AAOLM Issue Listing
4. timelinemap_view.pt			Topic		View for Topics to display Events w/ Timeline + Google Map
5. s3slider_view.pt             Folder      View for Folder that makes a slideshow out of a list of Photos
6. video_view.pt                Link        View for Links that renders links to YouTube and Vidmeo as videos


Customizations
==============

1. document_view.pt     Document    Modifies a standard document to look like an AAOLM article
2. event_view.pt        Event       Adds Google Map + Directions link to Event
3. document_byline.pt               Changed the byline to say 'By' instead of 'Contributors'
